CSS Working Group

The CSS Working Group. part of W3C. sets the standards CSS. The group meets on a regular basis to update the standards. Many of these are krw.vn as working drafts.

Instead of releasing CSS versions, su&i as CSS I and 2, the group has changed its approach for CSS updates. The CSS Working Group has broken CSS into modules to parts of CSS. The group now publishes CSS Snapshots, which include me latest updates to CSS. The group intends to publish these snapshots every one to two years. The latest CSS Snapshot is CSS Snapshot 2018.
